WebSphere

Windows安装WebSphere8.5.5

官网： http://www-01.ibm.com/support/docview.wss?uid=swg27004980

进行下载：https://www-01.ibm.com/support/docview.wss?uid=swg27025142#ibm-content

创建一个账号

应用这个安装工具 进行下载 还需要科学上网才可以

下载完成后 进行安装 一路默认即可

还要用WAS作为库 去支撑

官网： http://www.ibm.com/en-us/homepage-a.html

搜索这个：

Websphere Application Server for Developers

下载完成后

导入并全部解压

然后添加引导

然后进行安装

一路默认下去 进行安装

继续进行添加server

然后启动概要管理 工具

进行创建 这边我的选择是应用服务器

一路默认过去

这是一些默认的端口

这里的对勾去掉

一路默认 进行创建

成功创建 概要文件配置完成

概要文件的目录结构：

C:\Program Files (x86)\IBM\WebSphere\AppServer

管理员权限运行cmd 

cd C:\Program Files (x86)\IBM\WebSphere\AppServer\bin
startServer -help

启动WAS

startServer server1 -profileName AppSrv01

启动成功

停止

stopServer 服务名

本地进行访问

http://localhost:9060/ibm/console

尝试登录 登录成功

远程访问也是可以的

下载安装部署就完成了

docker安装WebSphere7

docker search WebSphere7
docker pull /websphere7
docker run -d -p 9060:9060 -p 9043:9043 -p 8880:8880 -p 9080:9080 iscrosales/websphere7
# 开启docker的websphere7

远程登录一下

成功部署完成
